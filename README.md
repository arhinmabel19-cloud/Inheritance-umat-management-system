# Inheritance-umat-management-system

This project demonstrates the concept of inheritance in Python using a simple UMaT Management System.
## Features
- Parent class: Person
- Child class: Student
- Child class: Lecturer
- It uses 'super()' to initialize inherited attributes.
- It demonstrates method inheritance.
- It demonstartes method overriding.
- It includes student enrollment and lecturer teaching methods.

## Classes
### Person
Attributes
- name
- age

Method:
- display_info()

### Student
Additional Attribute:
- student_id

Method:
- enroll_course()

### Lecturer
Additional Attribute:
- employee_id

Method:
- teach_course()
